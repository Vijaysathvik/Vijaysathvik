## 🛠️ Projects

**[layer10-memory](https://github.com/Vijaysathvik/layer10-memory)** · `Python` `SQLite` `D3.js`
> Grounded long-term memory system over Apache mailing list archives. Parses MBOX corpora, deduplicates artifacts via SimHash, extracts structured entities and claims using an LLM pipeline, and stores them in a queryable SQLite knowledge graph. Retrieval uses hybrid BM25 + embedding cosine ranking. Ships with a D3 force-graph visualisation with a timeline scrubber and evidence panel.

**[Food-delivery](https://github.com/Vijaysathvik/Food-delivery)** · `Node.js` `Express` `MongoDB` `React` `Stripe`
> Full-stack food delivery platform with a REST API backend (JWT auth, Stripe payments), a React user frontend, and a separate React/Vite admin dashboard for managing items and orders.
